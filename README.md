# DroneUp Vendor Portal

Vendor management portal for drone delivery partnerships with retailers.

## Overview

Admin dashboard for managing drone delivery operations across multiple vendor locations. Track orders, monitor store performance, and configure delivery settings.

## Key Features

- **Dashboard**: Real-time order metrics, sales analytics, and store maps
- **Transaction Management**: Search and filter orders by vendor, date, and status
- **Vendor Administration**: Configure stores, fulfillment options, and pricing
- **Multi-Location Support**: Manage multiple store locations per vendor
- **Order Tracking**: Monitor picking, delivery, and completion status

## Tech Stack

Web-based CMS for vendor partners and internal operations teams.
